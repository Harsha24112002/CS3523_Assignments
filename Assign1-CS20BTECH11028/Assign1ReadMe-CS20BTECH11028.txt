For compiling:
In the terminal, cd into the directory containing the source file(Assign1Src-CS20BTECH11028.c)
Place the input txt file in the same directory named "sampleinput.txt" ,it should contain 
the number of threads, point, number of points and set of points respectively.
run the following command:

gcc Assign1Src-CS20BTECH11028.c -lpthread

For running:
In the same directory enter,
 
./a.out
